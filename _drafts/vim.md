#vim

## entering and exiting vim

```bash
# enter
vim

# exit vim
:q
```

## moving around

h,j,k,l,w,b keys

```
j -> moves down
k -> modews up
```

```
h -> moves left
l -> moves right
```

```
w -> moves forward one word
b -> moves backwards one word
```

## modes

- Normal (esc)
- Insert (i)
- Visual (v)
- Visual-line (shift v)
